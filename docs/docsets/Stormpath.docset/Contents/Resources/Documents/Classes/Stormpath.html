Stormpath

public final class Stormpath: NSObject

Stormpath represents the state of the application’s connection to the Stormpath Framework server. It allows you to connect to the Stormpath Framework Integration API, register, login, and stores the current account’s access and refresh tokens securely. All callbacks to the application are handled on the main thread.
